# MinecraftDecompiler
A useful tool to decompile and deobfuscate Minecraft by CFR/FernFlower and Proguard(official)/SRG(Searge 1.13-)/CSRG(Bukkit,Spigot,Paper)/TSRG(Searge 1.13+)/Tiny(Intermediary, Yarn) mappings
